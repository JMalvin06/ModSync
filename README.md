# ModSync
This mod simplifies the sharing of custom mod packs for multiplayer by linking your mods folder to a GitHub repository.

![demo](https://raw.githubusercontent.com/JMalvin06/ModSync/main/preview/demo.gif)

## Safety and Security
This mod is intended for use with private servers. 
For your safety, only use connect to repositories that you can absolutely trust.
This mod directly downloads jar files from GitHub, which *could* include malware.

## License Disclaimer
To respect the hard work of other mod developers, please make sure to always check licenses of the mods you plan to share. Do not use ModSync to mass-distribute content without explicit permission.

## Usage
1. Create a GitHub repository containing your modpack
   * You can find a simple example [here](https://github.com/JMalvin06/tiny_modlist)
2. Open Minecraft with ModSync installed and add the repository link

## Compatibility
Currently only compatible with **Fabric 1.21.1**
