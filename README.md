# ModSync
This mod simplifies the sharing of custom mod packs for multiplayer by linking your mods folder to a GitHub repository.

![demo](https://raw.githubusercontent.com/JMalvin06/ModSync/main/preview/demo.gif)

## Usage
1. Create a GitHub repository containing your modpack
    * You can find a simple example [here](https://github.com/JMalvin06/tiny_modlist)
2. Open Minecraft with ModSync installed and add the repository link

## Compatibility
* Currently only compatible with **Fabric 1.21.1**
